TPO : Introduction au langage HTML

Remarques :

  1. Pour écrire votre première page web, tous ce que vous avez besoin est un navigateur pour visualier la page et un éditeur de texte tels que notepad,, vs code, ....
  2. Dans ce TPO on va utiliser les balises basiques de HTML surtout
    • Formulaire : Form, label, input, textarea, button, select, option, fieldset et legend
    • Tableau : table, tr, td, ...
  3. saisie des champs suivis par * est obligatoire.
  4. La saisie de la vine est semi-automatique (utiliser les balises datalist et option).
  5. Lorsqu'on soumet le formulaire le contenu sera envoyé à votre adresse mail.
